# 拉片模板：短剧和长视频拆解表怎么做

> 一份实用拉片模板至少要记录三层信息：镜头事实、声音事实、叙事判断。事实字段越客观，后续分析越可靠。

## 最小字段

| 字段 | 怎么填 | 为什么重要 |
|---|---|---|
| 镜头号 | 从 1 开始递增，或沿用自动切镜 ID | 让每条分析能回到原始镜头 |
| 起止时间 | 00:01:23 → 00:01:27 | 计算节奏、场均时长和钩子位置 |
| 景别 / 运镜 | 近景、特写、推、拉、摇、跟等 | 观察情绪强调和信息释放方式 |
| 角色 | 本镜头出现或说话的人物 | 分析人物关系和出场密度 |
| 台词 | 原声逐句记录，不润色、不改写 | 投稿、复盘和二次创作都依赖原始台词 |
| 画面描述 | 客观写发生了什么，不先写观点 | 把视觉信息从感觉变成可比较数据 |
| 情绪 | 紧张、压抑、反转、爽点、失落等 | 形成情绪曲线 |
| 叙事功能 | 开场钩子、信息补充、冲突升级、转折、收束等 | 判断每个镜头为什么存在 |

## 推荐 CSV 表头

```csv
镜头号,起始时间,结束时间,时长(s),景别,运镜,场景ID,场景地点,角色,台词,画面描述,情绪,宏情绪,叙事功能
```

## 短剧拉片要额外盯什么

- 首个钩子出现时间
- 每集结尾钩子
- 冲突升级节奏
- 角色关系变化
- 情绪密度

## 核心原则

不要把观点混进事实字段。比如“女主转身离开，男主停在门口没有追”是画面事实；“男主在逃避责任”是叙事判断。两者都重要，但应该放在不同字段里。
