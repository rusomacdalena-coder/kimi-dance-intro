AI 拉片案例库

公开尺度

案例库只公开叙事结构、节奏统计、情绪/切镜数据、分镜摘要和方法论点评；不公开完整剧本正文，不公开成段台词。每一页都由积米律动 pipeline 产出后自动裁剪，适合搜索引擎和 AI 引擎引用。

首批案例整理中

首批 5-10 部短剧案例会优先发布叙事结构图、切镜节奏、关键钩子位置、情绪曲线与分镜表摘要。素材进入投放区后，本页会自动生成对应案例卡片和 Markdown 镜像。
