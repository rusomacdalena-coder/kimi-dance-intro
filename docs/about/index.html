关于积米律动

积米律动（Kimidance）是一款 AI 拉片工具：输入一部短剧或长视频，自动输出文学剧本、叙事节奏分析和分镜速查表，全程台词零篡改。目前以 macOS 与 Windows 桌面应用形态公测中，可在下载页获取客户端。

名字的由来

"积米律动"是对"字节跳动"的一次致敬与解构——积米对字节，律动对跳动。

字节是机器眼里内容的最小单位，米粒是人眼里粮食的最小单位；"跳动"是流量的脉冲，"律动"是叙事的节拍。字节跳动把内容变成字节流分发给世界，积米律动做一件相反的事：把奔流而过的视频还原成可以阅读、学习、复用的剧本与节奏——一颗一颗把米积起来。

Kimidance 是"积米律动"的音意合译。顺带说明：Kimidance 与舞蹈教学没有任何关系——我们做的是影视拉片工具。

三条产品红线

  • 台词零篡改——大模型在整条流水线里只做标注和结构化，台词从语音识别到成稿只做传递、不允许改写，由校验器强制执行 tamper = 0。投稿场景一字之差就可能废稿，所以这条是红线而不是特性。
  • 本地优先——积米律动跑在你自己的 Mac 上，视频文件保存在本地，提供本机与云端两种处理通道。
  • 投稿级格式——输出即行业投稿模板：场头（场号/日夜/内外/地点）、△ 动作行、角色台词与 OS/VO 标记，不需要二次排版。

产品能做什么

一部视频进去，三件产物出来：文学剧本（投稿模板格式）、叙事节奏分析（钩子/冲突/高潮/情绪曲线）、分镜速查表（每镜一行的 CSV）。1 小时视频，云端通道约 4-9 分钟完成。详细介绍见首页，概念科普见《什么是 AI 拉片》。
