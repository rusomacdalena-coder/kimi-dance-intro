安装指引

公测期安装包暂未购买商店签名证书，首次打开时系统会提示"无法验证开发者"——这是未签名应用的正常提示， 并非安全问题。你可以用下方的 SHA-256 校验值确认安装包完整未被篡改，然后按以下步骤放行：

macOS

  1. 双击下载的 .dmg，把 Kimidance 拖入"应用程序"文件夹；
  2. 首次打开时若提示"无法打开，因为无法验证开发者"，点"完成"（不要点"移到废纸篓"）；
  3. 打开系统设置 → 隐私与安全性，滚动到底部，找到"已阻止 Kimidance"，点仍要打开；
  4. 再次确认后即可正常启动，之后不会再提示。

Windows

  1. 双击下载的 .msi 开始安装；
  2. 若出现 SmartScreen 蓝色提示框，点更多信息 → 仍要运行；
  3. 安装完成后从开始菜单启动 Kimidance。

校验安装包完整性（可选）

下载完成后可比对 SHA-256 校验值，确认文件与官方发布完全一致：

# macOS（终端）
shasum -a 256 ~/Downloads/Kimidance-Mac-0.2.2.8.dmg
# 应输出：2e94fb3012b3b12051d137d92c769b7409f1043b92baea75549a04baa70e1a73

# Windows（命令提示符）
certutil -hashfile "%USERPROFILE%\Downloads\Kimidance-Windows-0.1.0-x64.msi" SHA256
# 应输出：65ba4314f991f9b84c78b62e003312605ca88a60e4ea3d75a25f7e2ed617e595

常见问题

积分是什么？用完了怎么办？

积分是处理视频的计量单位，按视频时长消耗。注册赠送的积分足够体验完整流程；用完后可在客户端内通过微信扫码购买。

我的视频会上传吗？

视频在你的电脑本地处理。分析过程中仅将必要的关键帧与台词文本发送至 Kimidance 网关完成 AI 标注，不会上传完整视频文件。

支持 Intel 芯片的 Mac 吗？

当前 macOS 版本仅支持 Apple Silicon（M1 及以后）。Intel Mac 支持在计划中，欢迎邮件告诉我们你的需求。
